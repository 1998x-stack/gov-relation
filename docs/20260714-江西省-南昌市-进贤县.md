## 10. 数据文件说明

### 10.1 SQLite 数据库

- **路径**：`data/database/jinxian_network.db`
- **表结构**：

| 表名 | 记录数 | 说明 |
|------|--------|------|
| `persons` | 19 | 人物基本信息（姓名、籍贯、出生年月、学历、现任职务等） |
| `organizations` | 50 | 组织机构信息（名称、类型、层级、所属地区） |
| `positions` | 75 | 任职记录（人物→单位，含起止时间、级别） |
| `relationships` | 18 | 人物间关系（类型、交集单位/时期、上下文描述） |

### 10.2 使用示例

```sql
-- 查询某人的完整履历
SELECT p.name, pos.title, o.name, pos.start, pos."end"
FROM positions pos
JOIN persons p ON pos.person_id = p.id
JOIN organizations o ON pos.org_id = o.id
WHERE p.name = '熊振强'
ORDER BY pos.start;

-- 查询两个人物之间的关系
SELECT pa.name, pb.name, r.type, r.context
FROM relationships r
JOIN persons pa ON r.person_a = pa.id
JOIN persons pb ON r.person_b = pb.id;

-- 查询在某单位工作过的所有人
SELECT p.name, pos.title, pos.start, pos."end"
FROM positions pos
JOIN persons p ON pos.person_id = p.id
JOIN organizations o ON pos.org_id = o.id
WHERE o.name LIKE '%进贤%';
```
