## 1. 地理位置与基本概况

东兴市是广西壮族自治区防城港市代管的县级市，位于中国大陆海岸线最西南端，
东南濒北部湾，西南与越南芒街市接壤（是中国大陆唯一与越南海陆相连的国家一类口岸城市）。
东兴国家重点开发开放试验区是国家级重点开发开放试验区。

东兴市辖3个镇：东兴镇、江平镇、马路镇。

## 9. 调研局限性说明

本次调研遭遇到严重的网络访问障碍：
- **Exa (WebSearch)**: 免费 MCP 限流
- **Baidu Baike**: HTTP 403 禁止访问
- **Baidu Search**: 百度安全验证（验证码墙）
- **Google**: 自动化检测拦截
- **Bing**: 301 重定向后不可用
- **dongxing.gov.cn**: DNS 无法解析（NXDOMAIN）
- **fcgs.gov.cn**: 可访问，但领导之窗页面未显示县区级领导信息
- **web.archive.org**: 连接被拒绝
- **Jina Reader**: 连接超时

在此条件下，无法进行任何基于当前网络来源的调查。本报告基于预训练知识，
所有数据均标注为 `unverified`。
