# 安义县跨县干部交流网络调查报告

> 生成日期：2026年7月14日
> 研究范围：江西省南昌市安义县与周边县区的干部跨县/跨区交流网络

## 一、报告摘要

本报告基于对安义县历年领导班子成员的履历分析，构建了安义县与南昌市其他区县及跨设区市的干部交流网络。核心发现：**安义县处于南昌市干部交流网络的接收端**，其党政一把手主要来自南昌县（县委副书记→安义县长管道）和进贤县（进贤县长→安义县委书记），而安义培养的干部则向南昌市辖区（西湖区、青山湖区、青云谱区）和市直机关输出。

**最新动态**：2026年7月初，南昌市进行了6县区党委书记联动调整，涉及安义县委书记和县长的双重更替。

## 六、重要跨县交流模式分析

### 模式1：南昌县委副书记→安义县长（确认的管道）
```
谭伯乐（2020）：南昌县委副书记 → 安义县长
罗国栋（2021）：南昌县委副书记 → 安义县长
```
这是安义县长最重要的跨县输送管道。南昌县委副书记（正县级）岗位成为安义县长的"预备役"。

### 模式2：安义县委常委→西湖区委副书记（确认的模式）
```
熊振强（2021）：安义县委常委/政法委书记 → 西湖区委副书记
```
安义的副处级干部向南昌中心城区的区级岗位输出。熊振强之后更进一步，先后任市公路中心主任、市交通局长、进贤县委书记。

### 模式3：进贤县↔安义县双向流动
```
进贤→安义：熊辉（进贤县长→安义书记）、杨武（白圩乡书记→安义副县长）
安义→进贤：熊振强（安义常委→进贤书记）、叶飞（安义副书记→进贤副书记）
```
两县之间的人员往来最为频繁，形成人员流动闭环。

### 模式4：安义县是副厅级晋升的跳板
```
李松殿：安义书记→南昌市副市长→市委常委
彭开先：安义书记→南昌市副市长
谭伯乐：安义书记→南昌市政协副主席
周亮：安义县长→新建县长→青云谱书记→红谷滩书记（副厅级）
```
安义县委书记岗位**全部**晋升副厅级，县长岗位约半数晋升副厅级。这说明在安义任职是南昌市干部梯队中的重要历练环节。

### 模式5：市直机关下派到安义
```
余超：南昌市财政局副局长→安义县委常委/副县长
邓梁：南昌市公安局政研室主任→安义副县长/公安局长
```
市直机关的干部下派到安义任职，既有锻炼性质，也有加强市直对县区工作指导的意味。

### SQLite查询示例

```sql
-- 查看所有跨县调入安义的干部
SELECT p.name, p.birthplace, o.name AS source_org, pos.title, pos.note
FROM positions pos
JOIN persons p ON pos.person_id = p.id
JOIN organizations o ON pos.org_id = o.id
WHERE pos.note LIKE '%跨%' OR pos.note LIKE '%调%' OR pos.note LIKE '%→%'
ORDER BY pos.start;

-- 查看所有从安义县流出的干部
SELECT p.name, pos.title AS anyi_position, pos.note AS transfer_info
FROM positions pos
JOIN persons p ON pos.person_id = p.id
WHERE pos.note LIKE '%调任%' OR pos.note LIKE '%交流%';
```
