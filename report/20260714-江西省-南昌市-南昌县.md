## 6. ⭐ 南昌县↔进贤县核心通道 {#6}

### 这是南昌县干部输出的第一目的地

```
南昌县黄马乡 ──→ 进贤县副县长/常委
    │
    ├── 熊军（书记 2016-2021）──→ 进贤县副县长（2021.10-）
    │
    └── 徐永钢（乡长 2016-2019）──→ 进贤县副县长（2021.01）
                                    └── 进贤县委常委/组织部长（2023.05-）

南昌小蓝经开区 ──→ 进贤县
    │
    └── 聂红兵（招商局长 2020-2024）──→ 进贤县委常委/副县长（2024.02-）

南昌县/市系统 ──→ 进贤县（逆向流动，罕见）
    │
    └── 邓之武（进贤常务副县长）──→ 南昌县委副书记（2025.01）
                                    └── 东湖区委副书记/代区长（2026.07）
```

### 黄马线：最紧密的双人通道

| 维度 | 详情 |
|------|------|
| **熊军** | 南昌县人，黄马乡党委书记（2016-2021），2021.10起任进贤副县长 |
| **徐永钢** | 丰城人，黄马乡党委副书记/乡长（2016-2019），2021.01起任进贤副县长，2023.05升组织部长 |
| **共事** | 2016-2019年搭班子3年（书记+乡长） |
| **路径** | 先后调入进贤，间隔不到1年（2020.12 vs 2021.08） |
| **模式** | 南昌县乡镇主官→进贤县副处级领导 |

### 通道特征

- **单向流动**：南昌县→进贤县有 **4人**（熊军、徐永钢、聂红兵 + 邓之武曾在进贤任职），反向只有邓之武1人且是"路过"
- **无真正的双向交流**：进贤县几乎没有干部直接调入南昌县
- **解释**：南昌县是"江西经济第一县"（GDP 1401亿），进贤县经济体量较小——这是一种"强县输出、弱县接收"的非对称人才输送模式
