4. **山东籍但在江苏成长**：山东嘉祥人，但职业生涯主体在江苏完成。

## 4. 关键发现

1. **苏州→镇江的干部输出**：吴庆文从苏州市长调任镇江市委书记，延续了苏州干部向苏南其他城市输出的传统（类似杜小刚从苏州到无锡）。
2. **前书记马明龙的去向**：马明龙（2020年2月–2026年3月任书记，任期约6年）在2026年3月离任后去向待查。
3. **前市长徐曙海的去向**：徐曙海（2020年4月–2026年6月任市长，任期约6年2个月）在2026年6月离任后去向待查。
4. **吴庆文的"高校纪委→经济主官"特殊路径**：从哈尔滨工程大学、南航的纪委书记到苏州工业园区一把手（经济最前沿），再到苏州市长和镇江市委书记，其职业转型跨度大但极为成功。
5. **6区县信息不足**：仅确认3位区委/市委书记姓名，其余9位区县党政一把手均待查。
6. **市人大常委会主任**：陈可可（1968年4月生，江苏丹阳人，2026年1月当选）
7. **市政协主席**：郭建（1968年9月生，江苏扬中人，2022年3月当选）
