> 调查时间：2026年7月15日
> 覆盖时段：2020-2026年
> 关注重点：跨省/跨区域干部调任

## 一、调查概述

本报告系统梳理2020-2026年间广西壮族自治区省部级和重要厅局级干部跨省流动情况，涵盖省级党政一把手、党委常委、政府副职以及重要地市领导的跨省调任事件。数据来源包括百度搜索、政府官网、官方媒体（新华社、中新网、人民网、广西日报、澎湃新闻等）以及维基百科。

## 八、关键调任网络分析

### 8.1 "广西-山东" 对调模式 (2022年4月)
```
山东副省长 王心富 ────→ 广西党委常委（统战部长→组织部长）
广西党委常委 徐海荣 ──→ 山东省委常委
```
这一对调发生在两天之内，是典型的省际干部交流。

### 8.2 "广西-青海-河南" 三角流动 (2024年12月)
```
陈刚: 青海省委书记 ──→ 广西党委书记
刘宁: 广西党委书记 ──→ 河南省委书记
吴晓军: 青海省长 ────→ 青海省委书记
```

### 8.3 回归型调任 - 韦韬的独特路径
```
广西罗城人─→玉林市长─→广西北部湾港务集团董事长
    ↓ (2021.02 跨省)
山西副省长─→太原市委书记
    ↓ (2025.07 回归)
广西自治区主席
```

### 8.4 西藏-广西-宁夏 跨区系调动（何文浩）
```
国家安全部
    ↓
宁夏 → 西藏（政法委书记）
    ↓ (2021.10 调任)
广西（党委常委、政法委书记）
    ↓ (2024.07 免职)
```

## 九、调任模式总结

### 9.1 "省际对流" 特征
- **强对流省份**: 山东、山西、青海、河南、福建、海南
- **弱联系省份**: 广东、云南、贵州（地理邻近但高层调任少）
- **特殊对流**: 西藏、新疆（均为政法系统干部调入）

### 9.2 "中央-地方" 循环
- **中央→广西**: 中组部（王维平）、中纪委（苗庆旺）、质检总局（孙大伟）
- **广西→中央**: 全国人大（彭清华、陈武）、民政部（李常官）

### 9.3 时间集中性
- **2021年10-11月**: 大规模调整（刘宁、徐海荣、何文浩同时调入）
- **2024年12月**: "四省联动"调整（广西、青海、河南、湖北）
- **2025年7月**: 蓝天立被查后的应急接替

### 9.4 与广东的特殊关系
尽管广东与广西地理接壤且存在"粤桂协作"机制，但**省部级干部交流非常有限**。广西向广东的直接干部输出极少，这反映了粤桂间行政层级不对等（广东为经济大省，干部配置规格不同）。
