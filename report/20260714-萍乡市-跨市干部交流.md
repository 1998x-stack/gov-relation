## 二、已确认的萍乡跨市干部交换模式

### 2.1 模式一："赣西环线" — 宜春↔萍乡↔景德镇↔上饶↔宜春（颜赣辉闭环）

**核心人物：颜赣辉** — 这是赣西环线最典型的案例。

颜赣辉的完整路径：
```
宜春市委常委/副市长(2005.01–2011.08)
  → 萍乡市委副书记(2011.08–2013.09)        ← 萍乡节点
  → 景德镇市长(2013.09–2016.04)
  → 上饶市长(2016.04–2017.06)
  → 宜春市委书记(2017.06–2020.06, 后落马)    ← 回到起点
```

**确认的事实**：颜赣辉在萍乡担任**市委副书记**（2011.08–2013.09），约2年时间。这构成了"赣西环线"中萍乡段的明确连接。

**源**: https://zh.wikipedia.org/wiki/颜赣辉

### 2.2 模式二：景德镇↔萍乡↔宜春"旋转门"

| 人物 | 路径 | 性质 |
|------|------|------|
| **邓保生** | **景德镇市委书记**(2011–2013) → **宜春市委书记**(2013–2017) | **跨市平级调动** |
| **郭安** | 宜春→景德镇市委常委/常务副市长(2009–2011)→南昌→鹰潭 | 经景德镇中转 |

**说明**：邓保生从景德镇书记转任宜春书记，跨越了"景德镇—宜春"轴线。萍乡位于宜春西侧，与这一轴线地理相邻。

### 2.3 模式三：省直→萍乡"空降"模式

萍乡历史上多任市委书记为省直机关或外部调任：

| 书记 | 前任职务 | 来源 |
|------|---------|------|
| 余正琨(2026–) | 待查 | 外部调任 |
| 刘烁(2023–2026) | 省委政法委副书记（省直） | **省直空降** |
| 陈敏(2021–2023) | 江西省委统战部常务副部长（省直） | **省直空降** |
| 李小豹(2016–2021) | 团省委书记（省直） | **省直空降** |
| 刘卫平(2014–2016) | 省纪委副秘书长（省直） | **省直空降** |

**说明**：萍乡市长线也呈现类似特征。**刘烁**先后任市长(2021–2023)→书记(2023–2026)，路径与景德镇的陈克龙类似。

### 2.4 模式四：萍乡↔吉安（南部连接）

**最新案例**：
- **傅正华**（2026年5月任萍乡市长），籍贯吉安县。其从吉安体系调任萍乡，构成"吉安→萍乡"的南向通道。

### 2.5 模式五：跨省交流（萍乡↔湖南）

萍乡地处湘赣边界，西接湖南株洲、长沙。虽然搜索受限，但结合历史：
- 萍浏醴起义等历史渊源说明萍乡与湖南浏阳、醴陵有深层民间和产业联系
- 萍乡到长沙黄花机场约120km，到宜春明月山机场约60km，经济联系偏向湖南
- **待查**：是否有厅级干部在湘赣之间直接调任

## 三、周边城市交叉验证

### 3.1 张智萍（新余/宜春/吉安反复调动）

张智萍的路径（据新余报告）：
```
新余(分宜县) → 宜春(宜丰县长) → 吉安(遂川书记) → 新余(人大主任)
```
这是赣西三市（新余、宜春、吉安）之间罕见的跨市反复调动案例。与萍乡的关联：分宜县历史上曾属宜春、萍乡均有密切经济往来。

### 3.2 新余市的萍乡连接

新余报告确认：新余作为"接收型城市"，主要从省直空降（方向军、郑光泉、犹㼆均为省直来源），与萍乡的直接干部交流**极少**。新余与邻居宜春、吉安之间的平级干部交流同样较少。

## 四、颜赣辉路径的萍乡连接（详细）

### 关键时间线

| 时间 | 颜赣辉职务 | 涉及城市 |
|------|-----------|---------|
| 1982–2005 | 抚州→金溪县委→上饶市委组织部长 | 抚州、上饶 |
| 2005.01–2011.08 | **宜春市委常委、副市长** | **宜春** |
| **2011.08–2013.09** | **萍乡市委副书记** | **萍乡**（2年） |
| 2013.09–2016.04 | 景德镇市长 | 景德镇 |
| 2016.04–2017.06 | 上饶市长 | 上饶 |

| 2020.06 | 被查 | — |
| 2021.02 | 双开 | — |
| 2021.04 | 判刑11年（受贿约2610万） | 新余中院审判 |

### 萍乡在其网络中的角色

颜赣辉的萍乡任职（市委副书记）是其从宜春副市长到景德镇市长的 **"跳板"** 。在萍乡的2年副书记经历，帮助他完成了：
1. 从副厅级（宜春副市长）到正厅级（景德镇市长）的跨越
2. 从赣西（宜春/萍乡）到赣东北（景德镇/上饶）的地理区域切换
3. 积累地级市核心领导经验

这个路径成为"赣西环线"假说最有力的支撑。

## 五、跨市交流模式总结

| 模式类型 | 方向 | 活跃度 | 确认案例 | 说明 |
|---------|------|--------|---------|------|
| **赣西环线** | 宜春↔萍乡↔景德镇↔上饶↔宜春 | **高** | **颜赣辉**（已证实） | 最核心的跨区闭环 |
| **省直空降→萍乡** | 省直→萍乡 | **高** | 刘烁、陈敏、李小豹 | 萍乡是"空降型"城市 |
| **吉安→萍乡** | 吉安→萍乡 | **中** | **傅正华**（2026新人） | 新出现的南向通道 |
| 萍乡→景德镇 | 萍乡→景德镇 | **低** | 仅颜赣辉单一案例 | 通过副书记岗位中转 |
| 萍乡→新余 | 双向 | **很低** | 几乎无直接案例 | 新余报告确认交流极少 |
| 萍乡→宜春 | 双向 | **中** | 颜赣辉、张智萍（间接） | 受颜赣辉落马影响可能减弱 |
| 萍乡↔湖南 | 跨省 | **待查** | 无直接确认案例 | 地缘优势但公开证据不足 |
