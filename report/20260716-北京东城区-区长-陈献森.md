## 4. 职业生涯总结

### 4.1 职业路径特征

**公安出身 → 共青团 → 街道主政 → 援藏 → 区领导 → 区长**，是典型的北京基层成长型干部路径。

1. **公安法制根基**（1996–2004，8年）：在西城公安分局法制办公室起步，从科员升至正科级副处长，后任派出所副所长。
2. **共青团系统**（2004–2010，6年）：西城区团委副书记至书记，属于培养年轻干部的"蓄水池"通道。
3. **街道主政**（2010–2013，3年）：德胜街道工委书记兼主任，实权基层一把手。
4. **援藏经历**（2013–2016，3年）：拉萨市堆龙德庆县委书记，属于**重要政治履历**——援藏经历是提拔副局级的典型加分项。
5. **东城区副区长/政法委书记**（2020–2024）：从副区长上升至常委、政法委书记。
6. **区长**（2024.12/2025.01–至今）：担任正局级区长。

### 4.2 晋升时间线

```
1996.07 ─ 参加工作（22岁）
2004.01 ─ 副处级（共青团西城区委副书记，32岁）

2020.06 ─ 副局级（东城区委常委、副区长，48岁）

```

### 4.3 关键跨越分析

- **首次进入领导干部序列**：2004年任西城区团委副书记（副处级），时年32岁
- **正处级**：2006年任西城区团委书记（正处级），时年34岁
- **副局级**：2020年任东城区委常委、副区长（副局级），时年48岁——从正处到副局用了14年
- **正局级**：2024年任东城区区长（正局级），时年52岁

### 4.4 教育和专业成长

- 本科为**公安专业**（中国人民公安大学安全防范专业），奠定其早期公安司法背景
- 先后在**中国人民大学**（行政管理研究生课程进修班）和**北京大学**（公共管理硕士MPA）深造，表现出持续的在职学习和向公共管理方向的转型
