### 晋升路径分析

毛伟明的仕途呈现**典型的"苏系"干部 + 中央部委 + 省际交流**的三段式特征：

1. **江苏深耕期（1982-2013，31年）**：从企业（常州绝缘材料总厂）起步，逐步转入地方党政系统，依次在武进、泰州、省发改委、省政府秘书长等岗位历练，最终升任副省长。特点是长期在江苏一省，根基深厚。
2. **中央部委期（2013-2015，2年）**：调入工信部任副部长，虽时间不长但完成了从地方干部到中央视野的跨越。
3. **跨省交流期（2015至今）**：
   - 2015年"空降"江西任常务副省长（5年），与后来接任其江西省常务副省长之位的易炼红有交集
   - 2020年短暂执掌国家电网（10个月），是罕见的"央企一把手→省长"任职路径
   - 2020年调任湖南至今（5年半）

**关键洞察**：毛伟明是少数从央企一把手（国家电网）直接转任省长的案例，显示其深得中央信任。他也是少见的完整经历了"企业-地方-中央-央企-地方"全链条的现任省长。

### 途中两个特殊节点

- **2001年武进市委书记到2003年泰州市长之间近1.5年**：任常州市委常委、常务副市长（1998年武进撤市设区进程中的过渡）
- **2013年1月-10月同时担任江苏省副省长**：后调任工信部

## 7. 干部交流网络分析

毛伟明的跨省调动路径反映了高层级干部交流的典型模式：

```
江苏（31年）
  ↓ 提拔
工信部（2年）
  ↓ 空降
江西（5年）
  ↓ 央企
国家电网（10个月）
  ↓ 再任省长
湖南（5年+）
```

值得注意的是，毛伟明在江西省担任的是**常务副省长**（省级政府"二把手"），这一经历为他后来主政一省奠定了基础。江西作为邻省，其治省经验对湖南有直接参考价值。

与沈晓明的跨省路径类似但不重叠：
```
沈晓明： 上海(科教→副市) → 教育部 → 海南(省长→书记) → 湖南(书记)
毛伟明： 江苏(企业→省府) → 工信部 → 江西(常务副省) → [国家电网] → 湖南(省长)
```

两人均在**习近平主政过的浙江**出生（毛伟明衢州、沈晓明上虞），均为浙江籍高级干部在省外任职的典型案例。

## 8. 关键洞察与突破线索

### 🔴 高优先级

1. **毛伟明1982-1993年常州绝缘材料总厂早期经历**：11年从车间主任到厂长的具体升迁时间线缺乏精确记录。作为其仕途起点，这段时间的具体履历对理解其工业背景和人脉形成至关重要。

2. **毛伟明在江苏期间与现任湖南干部的可能交集**：毛伟明在江苏任职31年，湖南省部分领导干部亦有江苏工作经历，需要进一步核查是否存在"故旧关系"。

3. **"国家电网系"湖南干部网络**：毛伟明2020年曾短任国家电网董事长，国家电网在湖南有省级公司，可能存在其"旧部"。

4. **沈晓明与毛伟明的"浙籍"干部网络**：现任省委书记和省长均来自浙江，可研究湖南政坛是否形成"浙籍"干部群。

5. **省委常委中新面孔的背景挖掘**：刘红兵、郭灵计、王俊寿等新任常委的详细履历待查。
