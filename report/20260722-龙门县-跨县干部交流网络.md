# 龙门县跨县干部交流网络调查报告

> 调查日期：2026-07-22
> 数据来源：龙门县人民政府门户网站、惠州市人民政府门户网站、百度百科
> 信息时效：截至2026年7月22日
> 任务：跨县区干部交流模式分析——龙门县（广东省惠州市）

---

## 一、概述

龙门县位于广东省惠州市北部，东邻河源市东源县、连平县，南接惠州市惠城区、博罗县，西连广州市从化区、韶关市新丰县，北靠韶关市新丰县。作为惠州下辖的山区县（全县以山地丘陵为主），龙门县在惠州市干部交流格局中扮演着"接收端"角色——干部多数来自惠州市直机关或其他县区。

本报告追踪龙门县历任及现任主要领导的来源地与去向，分析惠州市内部的跨县区干部交流模式。

---

## 二、流入龙门县（从外县/市直调来）

> **核心模式**：龙门县作为山区县，主要领导多由**惠州市直机关**或**惠城区/惠阳区**调任，属于"市直→县"或"发达县区→山区县"的典型下派模式。

**完整调入路径**：
1. 1987-2007：惠阳县（后撤市改区为惠城区）基层工作——镇隆中学教师、二轻局科员、县委组织部、矮陂镇党委书记、河南岸街道书记
2. 2007-2021：惠州市直机关——市安全监管局科长→副局长→局长、市应急管理局局长（**正处级**）
3. **2021年7月**：从惠州市直**调任龙门县**，任县委副书记、代县长→县长
4. 2022年9月：升任龙门县委书记
5. 2025年2月：晋升副厅级，兼任惠州市政协副主席

**交叉特点**：刘洪添是典型的"市直下派"型干部——30多年职业生涯大部分在惠城区（原惠阳县）和惠州市直度过，仅在2021年首次到龙门县任职。他的整个职业生涯围绕惠州市区，龙门县是其第一个也是唯一一个县级主政岗位。

## 五、惠州市跨县区干部交流模式分析

### 5.1 龙门县在整个惠州市干部交流格局中的位置

```
惠州市直机关
    ↑↓
惠城区（市中心城区） ←→ 惠阳区
    ↑↓                    ↑↓
博罗县（经济较强县）     惠东县（沿海县）
    ↑↓                    ↑↓
    └────── 龙门县（北部山区县）
```

**龙门县的角色**：作为惠州市唯一的纯山区县，龙门县在干部交流中处于**接收端**：
- **"市直→县"**：刘洪添（市应急管理局→龙门县）、王洋（市委组织部→龙门县）
- **"县→市"**：段致辉（龙门县→惠州市副市长）
- **被动接收**：龙门县主要党政主官多从外部调入，本地成长并输出至市级的案例较少

### 5.2 识别到的交流模式

| 模式 | 案例 | 说明 |
|------|------|------|
| **市直下派（正处→县正职）** | 刘洪添（应急管理局长→龙门县长/书记）、王洋（组织部副部长→县长） | 龙门县书记/县长多从市直正处级岗位调任 |
| **在岗高配（副厅级兼县委书记）** | 刘洪添（市政协副主席兼龙门县委书记） | 龙门县委书记在岗晋升副厅级仍留任 |
| **县→市提拔** | 段致辉（龙门县长→惠州市副市长） | 龙门县长成为惠州市领导培养平台 |
| **东西部协作挂职** | 邬淑娴（龙门→贵州普安县） | 广东对口帮扶贵州 |
| **上级下派挂职** | 陈琳（县委副书记挂职）、李锋（副县长挂职） | 可能来自省直或市直机关 |

### 5.3 潜在交流对象（待查）

根据龙门县与周边县区的地理关系和人事常规，以下跨县区交流可能存在，但需进一步核实：

| 可能的交流 | 推测依据 |
|-----------|---------|
| 博罗县↔龙门县 | 两县接壤，博罗县经济更强，可能存在干部交流 |
| 惠城区↔龙门县 | 刘洪添是惠城区人且长期在惠城区工作，但惠城区→龙门县的直接交流案例尚不明确 |
| 惠东县↔龙门县 | 同为惠州下辖县，但地理距离较远，交流可能性较小 |
| 从化区（广州）↔龙门县 | 虽跨市但接壤，可能存在广州对口帮扶惠州的挂职交流 |

### 7.3 初步结论
1. **龙门县是干部"流入型"县区**：主要领导几乎全部从惠州市直或外县调入
2. **"市直→县"是最主要的流入通道**：书记和县长均来自市直机关
3. **输出渠道有限**：仅段致辉一人明确从龙门县长升任惠州市副市长
4. **县委书记高配副厅级**：刘洪添以市政协副主席身份兼任，在惠州属于特殊安排
5. **挂职体系完善**：既有接收上级挂职干部（陈琳、李锋），也向外输出挂职（邬淑娴→贵州）
6. **与相邻县区的直接交流案例不足**：博罗县、惠东县与龙门县之间的干部直接调动案例尚未确认
