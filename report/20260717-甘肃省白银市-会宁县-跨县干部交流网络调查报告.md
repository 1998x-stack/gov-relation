## 2. 确认的跨县区人事连接

### 2.1 ✅ 确认：许伟民 — 白银市副市长兼会宁县委书记（会宁县↔白银市核心桥梁）

**证据**：

白银市领导之窗（[来源](https://www.baiyin.gov.cn/ldzc/byszf/xwm/index.html)）明确记载：
> "许伟民，男，汉族，1984年6月出生，研究生学历，工程硕士，中共党员，现任白银市政府副市长、党组成员，会宁县委书记。"

**意义**：
- 许伟民同时居于**白银市政府领导班子**（副厅级）和**会宁县委一把手**（正处级），是会宁县与白银市之间最直接的制度性联系通道
- 这种"市领导兼县一把手"模式在甘肃省较常见，通常意味着该县在市级层面有较强的话语权
- 该模式可能是许伟民后续晋升副厅级的过渡安排

### 2.2 ✅ 确认：丁肃静 — 白银市委常委、组织部部长（会宁县干部教育联系）

## 3. 跨县交流模式分析

### 模式A：市领导兼县一把手（许伟民模式）

```
会宁县委书记 → 白银市副市长（兼任）
                  ↓
          可能的晋升路径：
          白银市副市长（副厅级）→ 市委常委/常务副市长等
```

这是最明确的跨级交流模式。许伟民同时占据城市和县级两个岗位。

### 模式B：县委书记跨县晋升（刘凯模式 — 参考案例）

刘凯（白银市委书记）的跨县路径：
```
灵台县县长（2011-2015）→ 灵台县委书记（2015-2019）
→ 平凉市委常委（2019-2021）→ 嘉峪关市长（2021-2026）
→ 白银市委书记（2026-）
```

此模式展示了甘肃省内跨县区、跨地市的干部交流路径。

### 模式C：区内逐级晋升（周春材/李晓清模式）

- 周春材：平川区委副书记 → 平川区长（区内晋升）
- 李晓清：靖远县长 → 靖远县委书记兼县长（区内一肩挑）

这两种都是"本地成长型"干部。

### 模式D：组织部长驱动的跨县区交流（丁肃静模式）

丁肃静作为**白银市委组织部部长**，她的岗位职责本身就涉及全白银市范围内的县处级干部交流调配：
- 会宁县 ↔ 靖远县
- 会宁县 ↔ 白银区
- 会宁县 ↔ 平川区
- 会宁县 ↔ 景泰县

这属于制度性（而非人际性）的连接。

## 4. 与定西市的潜在连接

会宁县南接定西市（安定区、通渭县等）。**目前未发现**会宁县领导与定西市之间的干部交流证据。

可能的原因：
- 定西市是独立的地级市，跨市交流比市内跨县交流少见得多
- 跨市交流通常发生在厅级干部层面，而非县处级
