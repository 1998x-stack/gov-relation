## 五、跨县区交流模式分析

### 5.1 已确认的模式
1. **"县长→县委书记"内部晋升**：龙门县呈现强烈的"县长升书记"模式
   - 范志益：县长 → 书记（2016）
   - 段致辉：县长 → 书记（2021）
   - 刘洪添：县长 → 书记（2022）
2. **兼任市级职务**：
   - 段致辉在任书记同时兼任惠州市副市长（2022年1月起）
   - 刘洪添在任书记同时兼任惠州市政协副主席（2025年2月起）

### 5.2 离任后的典型去向
| 人物 | 离开龙门后去向 | 类型 |
|------|---------------|------|
| 林洪 | 惠东县委书记 | 平调（县→县） |
| 范志益 | 大亚湾区区委副书记、区管委会主任 | 平调（县→开发区） |
| 段致辉 | 惠州市副市长（兼书记）→ 副市长 | 提拔（县→市） |
| 刘洪添 | 惠州市政协副主席（兼书记） | 提拔/转任（县→市政协） |

### 5.3 县委书记来源分析
- 刘洪添：来自市直机关（市应急管理局）→ 县长 → 书记
- 范志益：来自市直机关（市经信局）→ 县长 → 书记
- 段致辉：从县长内部晋升
- 林洪：调往惠东，属于跨县交流（惠东、龙门之间的人员互换模式不明）

## 八、关键发现总结

1. **"县长→书记"内部晋升传统**：龙门县连续三任县委书记均由县长升任，形成稳定传统
2. **市级职务兼任模式**：近两任县委书记（段致辉、刘洪添）均被加挂市级副职（副市长/政协副主席），显示该职位的重要性提升
3. **市级干部空降担任书记**：刘洪添和范志益均来自惠州市直部门（应急管理局/经信局），先任县长过渡再升书记
4. **空缺期隐患**：近十年两次出现县委书记长期空缺（2015年空缺7个月，2017年后亦有不短的空缺），反映县级主要领导人事安排的滞后
5. **典型跨县交流**：林洪（龙门→惠东）、范志益（龙门→大亚湾），龙门县的县委书记离任后多调往惠州市内其他区县或市开发区
