## 1. 核心发现：市长职位空缺

**截至2026年7月14日，新余市市长职位处于空缺状态，已超过3个月未任命新人选。**

### 空缺原因

市长空缺期间，市委常委、常务副市长**廖良生**主持市政府日常工作。市政府领导班子目前由7名副市长+1名秘书长组成：

## 3. 前市长徐鸿：因火灾问责被撤职

### 基本情况

| 项目 | 内容 |
|------|------|
| 姓名 | 徐鸿 |
| 性别 | 男 |
| 出生 | 1968年8月 |
| 籍贯 | 江西上饶 |
| 学历 | 博士研究生 |
| 市长任期 | 2021年4月 – 2024年7月 |

### 被免经过

- **2024年1月24日**：新余市渝水区佳乐苑临街店铺发生**特别重大火灾事故**，造成39人死亡、9人受伤
- **2024年7月5日**：新余市人大常委会接受徐鸿辞去市长职务
- **2024年9月**：国务院调查组发布调查报告，对55名公职人员进行问责，徐鸿被**撤销党内职务、政务撤职**

**徐鸿已成为江西官场因安全生产事故问责的典型案例。**

### 去向

被撤职后徐鸿未再担任任何公开领导职务，实际处于"落马"状态。

## 8. 周边区域人事交流模式

### "省直空降"主导模式

新余市作为江西省面积最小的地级市（3,178km²），其核心领导任职呈现明显的"省直空降"特征：

| 领导 | 前职 | 来源类型 |
|------|------|---------|
| 方向军 | 江西省统计局局长 | 省直空降 |
| 郑光泉 | 省生态环境厅党组书记 | 省直空降 |
| 犹㼆 | 省委副秘书长/省政府副秘书长 | 省直空降 |

**分析**：新余因经济体量较小、治理复杂度适中，被用作省直干部获取地方主政经验的"跳板"。这种模式在江西省较为典型——小市成为厅级干部"补地方课"的场所。

### 周边城市交流

- **张智萍**（现任人大主任）是罕见的跨市反复调动案例：新余(分宜县)→宜春(宜丰县长)→吉安(遂川书记)→新余(人大主任)
- **伍复康**（新余学院党委书记）与南昌县委书记贾彧超的团省委同期关系，是明确的**新余↔南昌**人事纽带
- 除此之外，新余与地理邻居宜春、吉安之间的**平级干部交流极少**

### 新余的角色判断

**接收型城市**（sink city）。省直干部大量"流入"获取地方经验，但"流出"到其他地市的案例较少。市长职位超过3个月空缺，进一步强化了这一判断——省里可能有更复杂的人事安排在酝酿中。
